# Modal

Custom Modal

This is a custom modal (not a native <dialog>). It can be moved by dragging the header and closed with the close button or by clicking the overlay. 
<section>
  <button type="button" class="hl-btn" id="open-modal">
    Open modal
  </button>

  <!-- modal overlay + box -->
  <div class="hl-modal-overlay hidden" id="hl-modal-overlay">
    <div
      class="hl-modal"
      role="dialog"
      aria-modal="true"
      aria-labelledby="modal-title"
    >
      <div class="heading">
        <div class="wrapper drag-handle">
          <div class="icon"></div>
          <p class="text" id="modal-title">Custom Modal</p>
        </div>
        <button class="hl-btn close no-drag" id="modal-close"></button>
      </div>
      <div class="content">
        This is a custom modal (not a native <dialog>). It can be
        moved by dragging the header and closed with the close button or
        by clicking the overlay.
      </div>
      <menu class="footer-btns">
        <button class="hl-btn" id="modal-ok">OK</button>
        <button class="hl-btn" id="modal-cancel">Cancel</button>
      </menu>
    </div>
  </div>
</section>
